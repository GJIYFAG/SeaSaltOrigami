# SeaSaltOrigami
Qt 卡牌游戏 - 海盐折纸

本版本根据参考截图继续调整了 UI 和规则：

- 左上角完整展示得分详情，重点突出总得分与本轮得分。
- 牌区左侧增加“已打出对牌”紧凑区域，便于查看本轮已经打出的对牌。
- 顶部右侧新增“展开菜单”按钮，移除中部左侧悬浮菜单按钮。
- 抽牌、弃牌、出牌动画改为按目标区域飞入：手牌、收集牌、增益牌、已打出对牌、弃牌堆分别对应自己的飞行动画目标。
- 海马牌改为 Canvas 绘制的海马图案，不再使用国际象棋马图标。
- 放手一搏逻辑修复：发起后立即结束当前玩家回合，其他玩家各完成一个最终回合后自动结算并重置新一轮。
- 轮次结算修复：本轮分、事件分、收集牌、增益牌、美人鱼均参与结算；结算后轮内卡牌重新洗回牌库并重新发牌。
- 结束本轮阈值从 10 分调整为 7 分。
